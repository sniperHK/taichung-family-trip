## 路線 1：台北 → 高美濕地

**Day 1 週五下午出發，目標 16:30 前抵達**

### 主要路線（建議）

```
台北 → 國道3號（南下）→ 中港系統交流道 → 國道4號（往清水方向）
→ 清水端下交流道 → 台17線（臨港路）→ 三順路 → 美堤街 → 高美濕地
```

- **距離**：約 180 km
- **平日順暢車程**：約 2 小時
- **週五下午車程**：約 2.5 ~ 3.5 小時（視出發時間）
- **建議出發時間**：13:00 ~ 13:30 離開台北（避開 17:00 後的下班車潮壅塞高峰）

### 替代路線 A：走國道1號

```
台北 → 國道1號（南下）→ 台中系統交流道 → 國道4號（往清水方向）
→ 清水端下交流道 → 台17線 → 三順路 → 美堤街 → 高美濕地
```

- **注意**：國1「后里 ~ 大雅」路段為全台前三大壅塞路段，週五下午極塞，不建議

### 替代路線 B：國道3號 + 台61（避開國4尖峰）

```
台北 → 國道3號（南下）→ 中港系統 → 國道4號 → 清水端
→ 台17線（臨港路）→ 接台61西濱快速道路 → 高美路 → 高美濕地
```

### 週五下午避車潮建議

| 策略 | 說明 |
|------|------|
| **提早出發** | 13:00 前離開台北最佳，14:00 後國3北部路段開始車多 |
| **避開國1** | 國1「后里 ~ 大雅」段是常態壅塞路段，走國3更順 |
| **國3易塞路段** | 「大溪 ~ 鶯歌系統」段假日易塞，提早走可避開 |
| **即時路況** | 出發前查 [1968 高速公路資訊網](https://www.1968services.tw/) 或撥打 1968 |

### 停車資訊

| 停車場 | 費用 | 備註 |
|--------|------|------|
| **高美濕地遊客中心停車場** | 小客車每次 50 元 | 最方便，可沖腳 |
| 高美路堤防旁停車場 | 每次 50 元 | 含免費沖腳 |
| 幼稚園旁空地 | 免費 | 位置較遠，步行稍久 |

- **重要**：假日高美路狹窄易塞車，建議走**三順路轉美堤街**路線抵達，避開高美路
- 2 月非旺季，較不會遇到交通管制，但週五傍晚人潮仍多
- 建議停遊客中心停車場，可騎 YouBike 或步行至木棧道

## 路線 2：高美濕地 → 震大金鬱金香酒店

**酒店地址**：台中市梧棲區築港路築港巷 66 號

### 路線

```
高美濕地 → 美堤街 → 三順路 → 台17線（臨港路）→ 築港路 → 酒店
```

- **距離**：約 7 km
- **車程**：約 15 分鐘
- **路況**：平面道路，傍晚離開高美濕地時可能遇到出場車潮，但影響不大

### 酒店停車

- **免費停車場**：飯店兩側有寬廣的戶外平面停車場，免費使用
- 車位充足，不需擔心

## 路線 3：酒店 → 台中海洋館

**海洋館地址**：台中市清水區海洋路 168 號

### 路線

```
酒店（築港路）→ 台17線（臨港路）→ 北堤路左轉 → 海洋路 → 台中海洋館
```

- **距離**：約 4 km
- **車程**：約 8 ~ 10 分鐘
- **路況**：週六早上順暢

### 停車資訊

| 停車場 | 費用 | 備註 |
|--------|------|------|
| **台中海洋館停車場** | 小客車每次 50 元 | 最近，從北堤路左轉海洋路即達 |
| 梧棲漁港第一停車場 | 每次 50 元 | 步行約 5 分鐘，車位更多 |
| 梧棲漁港第二停車場 | 每次 50 元 | 步行約 5 分鐘，備用選項 |

- 週六假日人潮多，建議 09:30 開館前抵達搶車位
- 若館內停車場滿，步行至漁港停車場很近

## 路線 4：台中海洋館 → 梧棲漁港

**漁港地址**：台中市清水區北堤路 30 號

### 路線

```
台中海洋館 → 海洋路 → 北堤路 → 梧棲漁港
```

- **距離**：約 1 km
- **車程**：約 3 分鐘（幾乎就在隔壁）
- 如果已停漁港停車場，步行即可，不用移車

### 停車資訊

| 停車場 | 費用 | 備註 |
|--------|------|------|
| **梧棲漁港第一停車場** | 每次 50 元 | 離漁市與餐廳最近 |
| 梧棲漁港第二停車場 | 每次 50 元 | 較遠但車位多 |

- **平日免費優惠**：清水區、梧棲區、沙鹿區居民憑身分證/駕照，平日（週一至週五）免費
- 2/21 是週六，需付費

## 路線 7：酒店 → 梧棲中港觀光夜市

**夜市地址**：台中市梧棲區文昌路 280 號（文昌路與文匯街口，梧棲中港高中旁）

### 路線

```
酒店（築港路）→ 大智路/文化路 → 文昌路 → 中港觀光夜市
```

- **距離**：約 2 km
- **車程**：約 5 分鐘

### 停車資訊

| 停車場 | 費用 | 備註 |
|--------|------|------|
| **夜市免費停車場** | 免費 | 機車約 200 格、汽車約 25 格 |
| 童醫院 B、C 停車場 | 依公告 | 步行約 8~10 分鐘到夜市 |

- **注意**：週五、六下午就會停滿！2/21 是週六，建議**早點出發**或停童醫院再走過去
- 夜市營業時間：每週二、四、五、六 16:30 ~ 23:30

## 路線 10：三井 Outlet → 大甲鎮瀾宮（回程順路選配）

**鎮瀾宮地址**：台中市大甲區順天路 158 號

### 路線

```
三井 Outlet → 臺灣大道十段 → 台17線（臨港路）→ 台1線（中華路）
→ 大甲區順天路 → 大甲鎮瀾宮
```

或走快速路線：

```
三井 Outlet → 台61（西濱快速道路，北上）→ 大甲交流道下 → 大甲市區 → 鎮瀾宮
```

- **距離**：約 18 km
- **車程**：約 25 ~ 30 分鐘
- 走台 61 較快且不塞車

### 停車資訊

| 停車場 | 費用 | 備註 |
|--------|------|------|
| 大甲區公所停三停車場 | 依公告 | 離鎮瀾宮最近（58 公尺） |
| 民營 Car in 光明站停車場 | 依公告 | 距離 279 公尺 |
| 文武停車場 | 依公告 | 距離 328 公尺 |
| 大甲體育場 | 免費（視情況） | 約 350 個停車位，步行 10 分鐘 |

- 鎮瀾宮在大甲市區內，周末人潮多，建議導航至附近停車場
- 逛完約需 30 ~ 60 分鐘

## 路線 11：大甲鎮瀾宮 → 台北（經大甲回程）

### 主要路線（建議）

```
大甲鎮瀾宮 → 132 縣道（甲后路）→ 國道1號后里交流道（北上）→ 台北
```

- **距離**：約 170 km
- **平日車程**：約 2 小時
- **週日下午車程**：約 2.5 ~ 3 小時

### 替代路線：走國道3號

```
大甲鎮瀾宮 → 台1線南下至清水 → 台17線 → 國道4號（東行）
→ 中港系統 → 國道3號（北上）→ 台北
```

- 繞路約 20 分鐘，但若國1后里段壅塞可考慮

### 替代路線：走台61接國3

```
大甲鎮瀾宮 → 台61（西濱快速道路，北上）→ 新竹南寮
→ 接國1或國3（北上）→ 台北
```

- 完全避開國道中部壅塞段
- 車程較長（約 3 ~ 3.5 小時），但不塞車時反而更快

## 路線 12：三井 Outlet → 台北（不經大甲直接回程）

### 主要路線 A（建議）：國道4號 → 國道3號

```
三井 Outlet → 台17線（臨港路）→ 國道4號（清水端上，東行）
→ 中港系統 → 國道3號（北上）→ 台北
```

- **距離**：約 185 km
- **平日車程**：約 2 ~ 2.5 小時
- **週日下午車程**：約 2.5 ~ 3.5 小時
- 國3「大溪 ~ 鶯歌系統」段週日下午北上易塞

### 主要路線 B：國道4號 → 國道1號

```
三井 Outlet → 台17線（臨港路）→ 國道4號（清水端上，東行）
→ 台中系統 → 國道1號（北上）→ 台北
```

- **注意**：國1「后里 ~ 大雅」段週日北上常態壅塞，盡量避免

### 替代路線：台61 西濱快速道路

```
三井 Outlet → 台61（西濱快速道路，北上）→ 新竹南寮
→ 國1竹北交流道 或 國3香山交流道 → 台北
```

- 台61 全線免收費，被稱為「窮人的高速公路」
- 適合國道嚴重壅塞時使用
- 車程約 3 ~ 3.5 小時，但免塞車壓力
- 路線沿海景觀佳

### 週日回程避車潮建議

| 策略 | 說明 |
|------|------|
| **早點回** | 12:30 出發，避開 15:00 後的北上車潮高峰 |
| **走國3** | 比國1順暢，避開國1「后里 ~ 大雅」壅塞段 |
| **國3備案** | 若國3「大溪 ~ 鶯歌系統」塞，下鶯歌系統走國2接國1 |
| **台61備案** | 國道全塞時，走台61到新竹再上國道 |
| **台74繞行** | 國3霧峰至中港系統塞車時：國4 → 台74 → 國3 |
| **即時路況** | [1968 高速公路資訊網](https://www.1968services.tw/) 或撥打 1968 |

## 重要提醒

1. **出發前必查**：[1968 即時路況](https://www.1968services.tw/)，週五下午 / 週日下午為國道尖峰
2. **Google Maps 即時導航**：出發後開啟即時路況，系統會自動建議替代路線
3. **國道4號是關鍵**：從清水端連接梧棲海線與國道系統的樞紐，善用國4可快速進出
4. **台61是救命路線**：國道全線壅塞時，台61免費且不塞車，沿海風景佳
5. **海線景點都很近**：Day 2 所有景點車程都在 10 分鐘內，不用趕
6. **高美濕地停車**：走三順路 → 美堤街路線，避開狹窄的高美路
